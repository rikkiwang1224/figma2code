# 统一样板间规范 (common-template)

统一样板间 **目录、分层、命名、代码/样式/图标规范**及严禁行为。

## 文件

| 文件 | 说明 |
|------|------|
| `SKILL.md` | 设计原则、目录与分层规范、拆分模式、命名、代码/样式/图标要求、严禁行为要点 |
| `reference.md` | 严禁行为完整条目、目录与分层示例代码 |
| `README.md` | 本说明 |

## 如何更新

1. 修改 `SKILL.md` 中的规则内容
2. 更新版本号（在 YAML frontmatter 中）
3. 更新示例代码（如需要）
4. 提交到 Git

## 版本历史

- **v1.0.0** (2025-01-XX): 初始版本
